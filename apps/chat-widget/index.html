Chat Widget Development

Development Mode

This page demonstrates the chat widget in development mode. The widget auto-initializes when the data-chat-widget attribute is present.

Environment

Make sure the API server is running on port 3001 (or the port specified in your .env file).

Quick Start

# Start the API server
cd apps/api
pnpm dev

# In another terminal, start the widget dev server
cd apps/chat-widget
pnpm dev

Usage Examples

HTML attribute:

<div data-chat-widget
     data-api-url="http://localhost:3001"
     data-conversation-id="optional-id">
</div>

JavaScript:

import { initChatWidget } from 'chat-widget';

const cleanup = initChatWidget({
  containerId: 'my-container',
  apiUrl: 'http://localhost:3001',
  onConversationCreate: (id) => {
    console.log('Conversation created:', id);
  }
});

// Later: cleanup();
